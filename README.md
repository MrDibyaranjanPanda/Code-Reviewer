# AI Code Review Assistant

An AI-powered Python code review platform that combines automated static analysis with intelligent AI-generated feedback.

The application allows users to create projects, upload Python files, analyze code using multiple code analysis tools, and view comprehensive code review results through a web interface.

---

## Features

- User registration and login
- JWT-based authentication
- Project creation and management
- Project deletion
- Python file upload
- Automated Python code analysis
- AI-powered code review
- Pylint code quality analysis
- Bandit security analysis
- Radon code complexity and maintainability analysis
- Review history
- Dashboard statistics
- PDF code review report generation
- Uploaded file deletion
- Project and review data management

---

```text
